Atlas User & Product Management API
Project Overview

This project is a FastAPI backend connected to MongoDB Atlas, designed to manage Users and Products. It provides REST endpoints with proper request validation, error handling, and is containerized using Docker.

✅ Features

FastAPI backend with RESTful APIs

MongoDB Atlas connection

User & Product management endpoints: create and fetch

Request validation with Pydantic models

Proper HTTP error handling (HTTPException)

Dockerized for easy deployment

API testing via Swagger UI & Postman

Version controlled using Git


🛠 Technologies Used

Python 3.11

FastAPI

Pydantic

MongoDB Atlas

Docker

Git & GitHub
